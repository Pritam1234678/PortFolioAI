# Portfolio Website (Vite + React + TypeScript)

Modern personal portfolio website built with React, TypeScript, and Vite.

## Tech Stack

- React
- TypeScript
- Vite
- CSS
- GSAP

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Install

```bash
npm install
```

### Run Dev Server

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Scripts

- `npm run dev`: Start development server
- `npm run build`: Type-check and create production build
- `npm run preview`: Preview production build locally
- `npm run lint`: Run lint checks

## Project Structure

- `src/components`: UI components
- `src/components/Character`: 3D character and scene logic
- `src/components/styles`: Section-wise styles
- `public/images`: Static images
- `public/models`: Encrypted model and HDR assets
